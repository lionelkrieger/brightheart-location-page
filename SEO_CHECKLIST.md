# Location Page SEO Checklist

Use this checklist when creating each new location page to ensure optimal SEO performance.

## Pre-Launch Checklist

### ✅ On-Page SEO

#### URL Structure
- [ ] Clean, descriptive URL (e.g., `/send-tutors-london/`)
- [ ] Lowercase only
- [ ] Hyphens for word separation
- [ ] No unnecessary parameters
- [ ] Location keyword included

#### Title Tag (Under 60 characters)
- [ ] Primary keyword at the beginning
- [ ] Location name included
- [ ] Brand name at the end
- [ ] Compelling and click-worthy
- [ ] Unique for each location

**Example**: `ADHD Tutors London - Award-Winning | Bright Heart`

#### Meta Description (Under 160 characters)
- [ ] Primary keyword included
- [ ] Location mentioned
- [ ] Value proposition clear
- [ ] Call to action included
- [ ] Phone number or key benefit
- [ ] Unique for each location

**Example**: `Expert ADHD tutors in London. Award-winning SEN tutoring with qualified teachers. Home visits & online. Call 0208 064 3800.`

#### Heading Structure
- [ ] One H1 tag only
- [ ] H1 includes primary keyword + location
- [ ] H2 tags for main sections
- [ ] H3 tags for subsections
- [ ] Logical hierarchy maintained
- [ ] Keywords in headings (natural, not stuffed)

#### Content Quality
- [ ] Minimum 800 words total
- [ ] 300-500 words unique, location-specific content
- [ ] Primary keyword in first 100 words
- [ ] Primary keyword density 1-2%
- [ ] Secondary keywords naturally integrated
- [ ] Location mentioned 5-10 times naturally
- [ ] No duplicate content from other pages
- [ ] Proper grammar and spelling
- [ ] Readable (Grade 8-10 reading level)
- [ ] Scannable (short paragraphs, bullet points)

#### Images
- [ ] At least 3 high-quality images
- [ ] Descriptive file names (e.g., `adhd-tutor-london.jpg`)
- [ ] Alt text with keywords (natural, descriptive)
- [ ] Compressed/optimized for web
- [ ] WebP format with fallbacks
- [ ] Lazy loading implemented
- [ ] Responsive images (srcset)

#### Internal Links
- [ ] Link to homepage
- [ ] Link to main services pages
- [ ] Link to other location pages
- [ ] Link to blog/resources
- [ ] Descriptive anchor text
- [ ] No broken links

#### External Links
- [ ] Link to Google Business Profile
- [ ] Link to social media profiles
- [ ] Link to relevant local resources
- [ ] All links open in new tab
- [ ] All links are working

### ✅ Technical SEO

#### Mobile Optimization
- [ ] Fully responsive design
- [ ] Touch-friendly buttons (min 44x44px)
- [ ] Readable font sizes (min 16px body)
- [ ] No horizontal scrolling
- [ ] Fast mobile load time (<3s)
- [ ] Passes Google Mobile-Friendly Test

#### Page Speed
- [ ] Load time under 3 seconds
- [ ] First Contentful Paint under 1.8s
- [ ] Largest Contentful Paint under 2.5s
- [ ] Cumulative Layout Shift under 0.1
- [ ] Time to Interactive under 3.8s
- [ ] Lighthouse score 90+

#### Schema Markup
- [ ] LocalBusiness schema implemented
- [ ] Organization schema
- [ ] BreadcrumbList schema
- [ ] Review/Rating schema (if applicable)
- [ ] Validated with Google's Rich Results Test
- [ ] No schema errors

#### Indexation
- [ ] Page is indexable (no noindex tag)
- [ ] Submitted to Google Search Console
- [ ] Included in XML sitemap
- [ ] Robots.txt allows crawling
- [ ] Canonical tag points to self
- [ ] No duplicate content issues

### ✅ Local SEO

#### NAP Consistency
- [ ] Name consistent across all pages
- [ ] Address format consistent
- [ ] Phone number format consistent
- [ ] Matches Google Business Profile
- [ ] Matches all citations

#### Location Signals
- [ ] City/town name in H1
- [ ] Region/county mentioned
- [ ] Local areas/neighborhoods listed
- [ ] Postcodes included
- [ ] Local landmarks mentioned (if relevant)
- [ ] Service area clearly defined

#### Google Business Profile
- [ ] Location page linked from GBP
- [ ] GBP category matches page content
- [ ] GBP description matches page
- [ ] Photos uploaded to GBP
- [ ] Posts created linking to page

#### Local Content
- [ ] Mentions local schools/councils
- [ ] References local partnerships
- [ ] Includes local statistics
- [ ] Features local testimonials
- [ ] Discusses local context/needs

### ✅ Conversion Optimization

#### Calls to Action
- [ ] Primary CTA above the fold
- [ ] Multiple CTAs throughout page
- [ ] CTA in footer
- [ ] Phone number clickable
- [ ] Clear value proposition
- [ ] Contrasting button colors

#### Trust Signals
- [ ] Awards/certifications displayed
- [ ] Reviews/ratings shown
- [ ] Testimonials included
- [ ] Credentials highlighted
- [ ] Professional photos used
- [ ] Security badges (if applicable)

#### Contact Information
- [ ] Phone number in header
- [ ] Phone number clickable (tel: link)
- [ ] Email address provided
- [ ] Contact form (optional)
- [ ] Business hours listed
- [ ] Multiple contact methods

### ✅ Content Strategy

#### Unique Content Elements
- [ ] Location-specific statistics
- [ ] Local testimonials
- [ ] Area-specific services
- [ ] Local tutor profiles (if applicable)
- [ ] Local partnerships mentioned
- [ ] Community involvement highlighted

#### Keyword Targeting
- [ ] Primary keyword identified
- [ ] Secondary keywords (3-5) identified
- [ ] Long-tail keywords (5-10) identified
- [ ] Keywords naturally integrated
- [ ] LSI keywords included
- [ ] Competitor keywords analyzed

#### Content Freshness
- [ ] Publication date added
- [ ] Last updated date added
- [ ] Plan for quarterly updates
- [ ] Seasonal content considered
- [ ] News/events section (optional)

## Post-Launch Checklist

### Week 1
- [ ] Submit URL to Google Search Console
- [ ] Request indexing
- [ ] Check for crawl errors
- [ ] Verify schema markup in search results
- [ ] Monitor page speed
- [ ] Check mobile usability

### Month 1
- [ ] Monitor rankings for target keywords
- [ ] Track organic traffic in Analytics
- [ ] Review bounce rate and time on page
- [ ] Check for broken links
- [ ] Monitor page speed trends
- [ ] Review conversion rate

### Quarterly
- [ ] Update statistics and data
- [ ] Add new testimonials
- [ ] Refresh images
- [ ] Update awards/certifications
- [ ] Review and update content
- [ ] Check competitor pages
- [ ] Analyze keyword performance
- [ ] Update schema markup if needed

## Common Mistakes to Avoid

- ❌ Duplicate content across location pages
- ❌ Keyword stuffing
- ❌ Thin content (under 500 words)
- ❌ Missing location-specific information
- ❌ No unique value proposition
- ❌ Slow page load times
- ❌ Not mobile-optimized
- ❌ Missing schema markup
- ❌ Inconsistent NAP information
- ❌ No internal linking
- ❌ Generic testimonials without location
- ❌ Stock photos instead of real images
- ❌ No clear CTAs
- ❌ Forgetting to submit to search engines
- ❌ Not tracking performance

## Success Criteria

A successful location page should achieve:

### Within 3 Months
- [ ] Indexed by Google
- [ ] Ranking on page 2-3 for primary keyword
- [ ] Generating organic traffic
- [ ] At least 1 conversion per week

### Within 6 Months
- [ ] Ranking on page 1 for primary keyword
- [ ] Ranking for 5+ secondary keywords
- [ ] Consistent organic traffic growth
- [ ] 5+ conversions per month

### Within 12 Months
- [ ] Top 3 position for primary keyword
- [ ] Ranking for 20+ keywords
- [ ] Featured snippet opportunities
- [ ] Local pack inclusion
- [ ] 10+ conversions per month
- [ ] Strong brand presence in local search

---

**Note**: SEO is a long-term strategy. Results typically take 3-6 months to materialize. Consistency and quality are key to success.
